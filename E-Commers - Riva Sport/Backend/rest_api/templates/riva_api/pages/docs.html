API Root

adalah titik awal atau endpoint utama dari sebuah aplikasi REST API. Ini adalah URL utama yang digunakan untuk mengakses semua sumber daya dan layanan dalam API Anda. API root biasanya digunakan sebagai basis untuk semua permintaan API.
  • Daftar Api Root
    GET/api/
    Mendapatkan semua data
    Respon Sukses
    HTTP Status 200 - Ok

    API root mengembalikan data yang sah dan tidak memerlukan autentikasi.

Pendaftaran

Pendaftaran akun dalam konteks aplikasi REST API adalah langkah kunci dalam memulai penggunaan platform kami yang inovatif. Melalui proses pendaftaran ini, pengguna dapat memperoleh akses eksklusif ke berbagai sumber daya dan layanan yang kami tawarkan, membuka pintu untuk berbagai peluang integrasi dan berinteraksi dengan API kami secara efisien.
  • Pendaftaran Akun
    POST/api/account/register/
    Mendaftarkan akun user sesuai dengan data yang user kirimkan melalui parameter
    Parameter Query
    first_name REQUIRED TYPEstring

    Berisi nama depan anda

    last_name REQUIRED TYPEstring

    Berisi nama belakang anda

    Note : Terkadang anda harus mengganti kombinasi nama depan dan belakang, karena nama mungkin sudah digunakan oleh orang lain.

    email REQUIRED TYPEemail

    Berisi email anda

    username REQUIRED TYPEstring

    Berisi Username untuk akun anda

    Note : Pastikan username yang anda masukkan UNIQUE

    password REQUIRED TYPEstring

    Berisi password untuk akun anda

    password2 REQUIRED TYPEstring

    Berisi verifikasi dari password yang anda masukkan. Pastikan isi dari password2 sama dengan password.

    Respon Sukses
    HTTP Status 201 - Created

    Pendaftaran mengembalikan data berupa informasi akun yang telah berhasil dibuat.
